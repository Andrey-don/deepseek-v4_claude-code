# Режимы работы: DeepSeek V4 + Claude Code

## Быстрый старт
Все скрипты в папке scripts/. Запускай двойным кликом.

---

## Режим 1 — Только DeepSeek V4 Pro

**Когда:** обычный код, рефакторинг, объяснения. Максимальная экономия — Claude не расходуется.

**Запуск:** scripts\mode1-deepseek-only.cmd

---

## Режим 2 — Claude Sonnet 4.6 (оркестратор) + DeepSeek V4 (исполнитель)

**Когда:** средние проекты, Claude планирует — DeepSeek пишет код.

**Запуск:** scripts\mode2-claude-lead.cmd

Откроются два терминала. Скажи Claude:
> Для рутинного кода используй DeepSeek: claude-ds -p "задача"

---

## Режим 3 — DeepSeek V4 (оркестратор) + Claude Sonnet 4.6 (эксперт)

**Когда:** большие проекты, DeepSeek управляет — Claude только для сложной логики.

**Запуск:** scripts\mode3-deepseek-lead.cmd

Скажи DeepSeek:
> Сложную архитектуру делегируй Claude: claude -p "задача"

---

## Сравнение режимов

| Режим | Стоимость | Когда |
|-------|-----------|-------|
| 1 — Только DeepSeek | Минимум | 80% задач |
| 2 — Claude ведёт | Средняя | Средние проекты |
| 3 — DeepSeek ведёт | Низкая | Большие проекты |
