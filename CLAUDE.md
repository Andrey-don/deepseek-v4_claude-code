# Связка Claude Sonnet 4.6 + DeepSeek V4

## Модель-исполнитель: DeepSeek V4

Для экономии токенов делегируй простые задачи DeepSeek через bash-команду:

    deepseek "твоя задача"

Или через free-claude-code прокси:

    claude-ds -p "твоя задача"

## Когда делегировать DeepSeek

Делегируй DeepSeek если задача:
- Написать или отрефакторить код (рутина)
- Исправить баг с понятной причиной
- Написать тесты
- Написать документацию
- Переименовать переменные / привести к стилю
- Разобрать структуру файлов проекта
- Написать SQL, конфиги, YAML

## Когда НЕ делегировать (решай сам)

- Архитектурные решения
- Безопасность и авторизация
- Сложная многошаговая логика
- Рефакторинг с изменением интерфейсов
- Работа с секретами и переменными окружения

## Схема работы в VS Code

1. Открыть два терминала через Split Terminal (кнопка справа сверху в панели терминала)
2. Левый терминал: deepseek (DeepSeek TUI) или claude-ds (free-claude-code)
3. Правый терминал: claude (Claude Sonnet 4.6 - оркестратор)

## Примеры делегирования

    deepseek "напиши функцию валидации email на Python"
    deepseek "добавь обработку ошибок в файл api/routes.py"
    deepseek "напиши unit-тесты для класса UserService"
    claude-ds -p "отрефакторь функцию calculate_roi в calculator.js"
